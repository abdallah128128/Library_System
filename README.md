# Library_System
# consol Application Using c++
It is a library system that consists of two main components: books and users.
You can do the following:
- add_book
- search_books_by_prefix
- print_who_borrowed_book_by_name
- print_library_by_id
- print_library_by_name
- add_user
- user_borrow_book
- user_return_book
- print_users
